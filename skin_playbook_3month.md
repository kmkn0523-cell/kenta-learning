## STEP 1:アカウントの再設計(最初の1日でやる)

### 🔹 ユーザー名(@の後)

覚えやすく、ジャンルが伝わるもの。**カタカナ or ローマ字 + ジャンル**が鉄板。

悪い例:`@abc_1234`(誰か分からない)
良い例:`@skin_reset_jp` / `@niki_naoshi` / `@hadaarea_otona`

### 🔹 表示名(アカウント名)

**名前 + 肩書 + 実績要素**の構成が最も反応が良い。

パターン例:
- `けん | 200万円の肌荒れ失敗から学んだ`
- `ハダキ | 大人ニキビを内側から治す`
- `Soma | 10年の肌荒れを克服・30代の肌リセット`

### 🔹 プロフィール文(ここが最も重要)

以下の**4行構成**に当てはめる。

```
【1行目】何者か+実績
【2行目】このアカウントで発信していること
【3行目】信頼を裏付ける要素
【4行目】お願い・導線
```

#### 完成例

```
10年の大人ニキビ・慢性肌荒れを内側から克服/美容に200万使った失敗体験者

外側ケアで治らなかった人に「本当に効いた"食事・腸・睡眠"の整え方」を毎日投稿

医師ではありません。あくまで体験ベースの発信です

noteに全工程を15,000字でまとめました→固定ポスト
```

### 🔹 ヘッダー画像

顔出しなしなら、**テキスト主体のバナー**がおすすめ。Canva で5分で作れる。

例:
- 黒地に白文字「スキンケアでは、治らない。」
- 中央に小さく「大人ニキビと慢性肌荒れの内側改善 / 200万円の失敗で辿り着いた答え」

### 🔹 アイコン

顔NGなら、以下のどれか:
- **手書き風イラスト**(肌・葉・シンプルモチーフ)
- **文字アイコン**(自分の名前1文字 or ジャンルの頭文字)
- **抽象モチーフ**(円・三角など)

絶対NG:フリー素材の写真、AIっぽい美女イラスト、ロゴジェネレータの機械的な出力

## STEP 2:固定ポストを作る(2日目にやる)

固定ポスト=プロフをクリックして最初に読まれる"自己紹介投稿"。ここがあるかないかで、フォロー率が3〜5倍変わります。

### 固定ポスト完成版(コピペ可)

```
はじめまして。

10年以上、大人ニキビと慢性肌荒れに悩みました。
美容に使った総額は200万円を超えました。

高級化粧品、美容皮膚科、SNSで話題の成分、レーザー、エステ。
どれも一時的で、根本は変わりませんでした。

変わり始めたのは、
スキンケアを減らして、
食事・腸・睡眠を整え始めてからです。

このアカウントでは、
同じ遠回りをしてほしくない人に向けて、
「内側から肌を整える方法」を発信しています。

医師ではありません。
一人の当事者が試して効いたことを、
誠実に書いていくだけです。

▼ 全工程をまとめたnoteはこちら
(リンクまたは「現在準備中」)
```

**運用ポイント**:
- noteが未公開の間は「現在準備中」でOK
- 毎月1回は固定ポストを更新して鮮度を保つ

## STEP 3:投稿の7つの型(これだけ覚える)

肌荒れジャンルで反応が取れる型を厳選しました。**毎日どれか1〜3本**投稿する。

### 型①:共感前後比較型(最強)

```
昔の僕:
・朝起きて鏡を見るのが嫌
・肌が気になって外出が億劫
・新作化粧水を試すたびに期待しては裏切られる

今の僕:
・鏡を見ることに抵抗がない
・すっぴんに近い状態で外に出られる
・化粧品の新作を追いかけなくなった

変わった理由:外側をやめて内側を整えたから。
```

### 型②:失敗告白型

```
肌荒れに悩んでた頃、よかれと思ってやってた間違い。

・1日3回の洗顔
・高い美容液を重ね塗り
・ニキビ箇所にクリームを塗り込む
・甘いヨーグルトを"善玉菌"として毎日
・サラダチキンとプロテインばかり

全部、肌を悪化させてた。
気づくのに8年かかった。
```

### 型③:逆張り・反常識型

```
肌荒れを治したいなら、
スキンケアを「足す」より「引く」。

足すべきは:
・水(甘い飲み物を減らす)
・睡眠(1時間早く寝る)
・発酵食品(腸を整える)

引くべきは:
・高級美容液
・過剰な保湿
・夜更かし
・小麦と砂糖

逆をやると、肌は静かに整い始める。
```

### 型④:リスト型(保存されやすい)

```
肌が変わった8つの習慣。

1. 朝日を5分浴びる
2. 水を1日2L飲む
3. 緑茶を常備する
4. 小麦を週3→週1に
5. 夜8時以降食べない
6. 化粧水は1本だけ
7. 亜鉛を1日15mg
8. 就寝前スマホ30分カット

どれも無料か、500円以下。
化粧水に3万払うより、ここ。
```

### 型⑤:問いかけ型(コメントが付きやすい)

```
「肌荒れが治らない」と言ってる人の9割は、
実はどれか当てはまってます。

□ 甘い飲み物を毎日飲んでる
□ 就寝が日付変わる頃
□ パン・麺が主食
□ ストレスで食いしばり癖
□ 便秘または下痢がち

何個当てはまりましたか?
```

### 型⑥:タイプ別診断型(note伏線)

```
大人ニキビが治らない人、ほぼ4タイプ。

A:皮脂過多型(Tゾーンテカる・甘いもの好き)
B:腸荒れ型(パン主食・便通不調)
C:ホルモン乱れ型(夜更かし・生理前悪化)
D:ストレス型(頬の赤み・食いしばり)

タイプによって、優先順位が全く違います。
一括りの情報は、誰にも効きません。
```

### 型⑦:ストーリー型(長文・スレッド)

```
【2年悩んだ頬のニキビが消えた話】

きっかけは、ある日YouTubeで見た動画でした。

半信半疑で始めたのが、
夜8時以降食べない「16時間ファスティング」。

最初の3日は正直つらかった。
でも4日目、朝の頭が異様にスッキリすることに気づいた。

1週間続けたら、便通が明らかに変わった。

2ヶ月後、周りから「肌変わったね」と言われた。
```

## STEP 5:エンゲージメント戦略(一番効くけど誰もやらない)

### 🔥 「与える→受け取る」の順番を守る

**1日30分、他のアカウントに丁寧なリプ**をつける。これが最速でフォロワーを増やす方法です。

#### やり方

1. **自分と同じジャンルのアカウント(フォロワー3,000〜50,000程度)を10個見つける**
2. **毎日1〜2回、その人たちの投稿に3行以上のリプ**をつける
3. リプの内容は「自分の体験 or 付加情報」。「いいですね!」「勉強になります!」のコメントは無価値
4. 自分のフォロワー外の人からも見られる → プロフィールに飛ぶ → フォローされる

#### リプの好例

❌ NG:「いいですね!勉強になりました!」

⭕️ OK:「これ、めちゃくちゃ共感します。自分も昔は高級化粧水に月3万かけてましたが、朝食抜きに変えたら2ヶ月で肌が落ち着きました。外側より内側って本当ですよね」

**1日30分のリプ**で、投稿ゼロの日でも毎日10〜30人に発見されます。

### 🔥 RT・引用RTの賢い使い方

- **自分が100%同意できる投稿だけ**RT
- 引用RTは「自分の切り口」を必ず添える
- 過度なRT連発はタイムラインを汚すのでNG(1日3回まで)

## STEP 6:アルゴリズム対策(Xで伸ばすための原則)

X(旧Twitter)のアルゴリズムは以下の行動を評価します:

### 📈 評価される行動

- **投稿から最初の15分の反応率**(初速が最重要)
- **リプライの付いた投稿**(会話が成立してる=良質)
- **プロフィールクリック率**(自分のアカウントへの興味)
- **フォロワー以外からの反応**(発見可能性が高まる)
- **滞在時間(長文が最後まで読まれる)**

### 📉 評価が下がる行動

- **外部リンク付きの投稿**(ツイート単体での反応が激減。noteリンクは要リプ欄)
- **ハッシュタグの付けすぎ**(3個まで。それ以上はスパム判定)
- **短時間での連投**(スパム扱い)
- **急激にフォロー数を増やす**(自分からのフォローの話)

### 🎯 実践的な対策

1. **投稿は1時間以上空ける**
2. **リンクは必ずリプに分ける**(例外なし)
3. **投稿後15分は返信・引用RTに反応する**
4. **ハッシュタグは1〜2個まで**
5. **長文(スレッド)は読ませる設計で書く**

## ⚠ 絶対にやらないこと

- ❌ **フォロバ目的の無差別フォロー**(後で痛い目を見る)
- ❌ **バズりたくてキャラを変える**(信頼が一瞬で崩れる)
- ❌ **他人の投稿のパクリ**(必ずバレる)
- ❌ **短期的な煽り投稿**(長期の信頼を犠牲にする)
- ❌ **フォロワー買い・いいね買い**(アルゴリズムに弾かれる)
- ❌ **数字に一喜一憂**(継続こそが最強の施策)

## 💬 メンタルの整え方

3ヶ月は長く感じるかもしれません。途中、伸びない期間が必ず来ます。

そのときに思い出してほしいこと:

> **「見られていない」のは、まだ何も始まっていないだけ。**

3ヶ月後、いまの投稿数の90日分が自分のアカウントに貯まっています。それが1人でも誰かに刺さったときに、そこから雪だるま式に広がるのがSNSです。

1日目と100日目の自分は、絶対に違います。
