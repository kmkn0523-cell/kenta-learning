# 毎食後に眠くなる人は、肌荒れも起きやすい。血糖値スパイクの話。

「昼食後にどうしても眠くなる」「夕方に甘いものが欲しくなる」

これは血糖値が急上昇→急降下しているサインです。そしてこの血糖値スパイクが、肌荒れの大きな原因になっています。

---

## 血糖値スパイクが肌を壊す2つのルート

**ルート1：インスリン → 皮脂過剰**
血糖値が急上昇するとインスリンが大量に出ます。インスリンは皮脂腺を直接刺激し、皮脂が過剰分泌されます。毛穴詰まり・ニキビの最大の原因です。

**ルート2：糖化 → 肌の老化**
余った糖がコラーゲンと結合し、変性させます。これが「糖化（グリケーション）」。糖化が進むと肌は黄ばみ、ハリを失い、シワが定着します。生み出される老化物質を「AGEs（終末糖化産物）」と呼びます。

## 食べる順番だけで血糖値は変わる

何を食べるか、より「どの順で食べるか」のほうが影響が大きい。

正しい食べ順：
1. 汁物・スープ（最初に胃に入れる）
2. 野菜・きのこ・海藻（食物繊維で糖の吸収を遅らせる）
3. 肉・魚・卵（タンパク質）
4. 米・パン・麺（最後）

この順で食べると、同じメニューでも血糖値の上がり方が30〜50%緩やかになります。

## 主食を変えるだけでも効く

- 白米 → 玄米・もち麦入り（食物繊維3倍）
- 食パン → 全粒粉パン・ライ麦パン
- うどん → 蕎麦（特に十割蕎麦）
- 砂糖入り飲料 → 水・お茶・ブラックコーヒー

砂糖入り飲料（カフェラテ・スポーツドリンク・甘いコーヒー）は、固形物より血糖値が上がりやすい。これをやめるだけで2週間で肌が変わる人もいます。

## 食後10分の散歩が最強

食後10〜15分のウォーキングは、血糖値の上昇を**20〜30%抑える**ことが研究で示されています。

立って洗い物するだけでも効果あり。「食べたら座らない」を意識するだけで違います。
