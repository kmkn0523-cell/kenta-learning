# 「貧血じゃない」と言われた女性ほど、実は鉄不足。フェリチンの話。

「健康診断で貧血なしって言われた」

## ヘモグロビンとフェリチンの違い

健康診断で測る「ヘモグロビン」は、**今血液中にある鉄**の量。

「フェリチン」は、**体内に貯蔵されている鉄**の量。**鉄の貯金**みたいなもの。

順番で言うと：
1. まず貯金（フェリチン）が減る
2. 貯金が尽きる
3. ようやくヘモグロビンが減る = 貧血と診断される

つまりヘモグロビンが正常でも、**フェリチンが空っぽ**の状態がある。これが「隠れ貧血」「**潜在性鉄欠乏**」と呼ばれる状態です。

## フェリチン理想値は50以上

基準値の解釈：

- 12 ng/mL以下 → 貯金ゼロ、緊急
- 12〜30 ng/mL → 不足、症状あり
- 30〜50 ng/mL → 要対策
- **50 ng/mL以上 → 肌・髪の健康ライン**
- 100 ng/mL以上 → 理想

検査値が「正常範囲（12以上）」となっていても、**肌の改善には足りない**。50を目指す必要があります。

## 隠れ貧血の症状チェック

3つ以上当てはまれば疑いあり：

- 朝起きるのがつらい
- 階段で息切れ
- 顔色が青白い or 黄色っぽい
- 髪が抜けやすい
- 爪が薄い・割れやすい・スプーン状に凹む
- 氷を食べたくなる（**氷食症**、典型症状）
- 立ちくらみ
- ニキビ跡が消えない
- 脚がムズムズする（**むずむず脚症候群**）

特に「氷食症」「むずむず脚」は鉄不足のかなり強いサイン。

## 検査の受け方

内科クリニックで「フェリチン値を測ってほしい」と伝える。**保険適用**で症状があれば1,000〜2,000円程度。

検査までの流れ：
1. 内科を予約
2. 「肌荒れと疲労感の原因として、フェリチンを測りたい」と伝える
3. 採血（5分）
4. 結果は1週間後

健康診断ではフェリチンを測らないことが多いので、自分から言わないと出てきません。
