## 🌱 販売14日前:種まき投稿(体験ストーリー型)

*まだ販売の話はしない。自分の過去の失敗談で共感を作るフェーズ。*

### バリエーション1

```
20代の頃、肌のために200万円使いました。

高い化粧水、美容液、最新成分、美容皮膚科のレーザー、エステ、サプリ……。

でも、どれも一時的でした。

肌は"結果"であって、"原因"じゃない。
外側ばかり整えても、中が乱れていれば、また荒れる。

今ならそう言い切れるけど、気づくのに10年かかった。
```

### バリエーション2

```
10年以上悩んだ大人ニキビが、
ある習慣を変えた数ヶ月で落ち着き始めた時、
やっと気づいた。

「肌は食べたものと眠った時間でできている」

高い化粧品ではどうにもできなかったことが、
食事と睡眠を整えただけで変わった。

この順番、逆だった。
```

### 運用メモ
- 投稿時間:**朝7時台 or 夜21時台**(この層が読む時間帯)
- 画像なし or 経過写真のごく一部(横顔・部分)
- コメントには丁寧に返信 → エンゲージメントを上げてアルゴリズム強化

## 🌱 販売7日前:種まき投稿(タイプ診断チラ見せ型)

*noteの一部を先に見せて「続きが知りたい」状態を作る。*

### バリエーション1(問いかけ型)

```
大人ニキビが治らない人、ほぼ4タイプのどれかです。

A:皮脂過多型(Tゾーンが常にテカる、甘いものが好き)
B:腸荒れ型(便秘/下痢、パン・麺が主食)
C:ホルモン・睡眠乱れ型(寝るのが遅い、生理前に荒れる)
D:ストレス炎症型(頬の赤み、食いしばりがある)

タイプによって、やるべき順番は全く違います。
```

### バリエーション2(断定型)

```
「肌荒れに良いらしい」を全部やると、だいたい失敗します。

皮脂過多型に必要なのは糖質カットと亜鉛。
腸荒れ型に必要なのはグルテンフリーとファスティング。
睡眠乱れ型に必要なのは就寝時間と朝日。
ストレス型に必要なのはオメガ3と自律神経ケア。

自分のタイプを知らずに片っ端からやるから、効果が薄れる。
```

### 運用メモ
- このツイートは**予告ポスト(5日前)の伏線**として機能する
- 「詳しく知りたい」「続きが気になる」のコメントが来たら、「来週noteで詳しく書きます」と軽く示唆

## ⚠ やってはいけない投稿NG集

- ❌ 「バズらせてください🙏」「拡散お願いします🙏」(一気に冷める)
- ❌ 絵文字とエクスクラメーションの多用(30代後半〜40代層には逆効果)
- ❌ 「今だけ!」「急いで!」の煽り口調(信頼を損なう)
- ❌ 同じ内容を1日2回以上投稿(告知疲れを起こす)
- ❌ 他の発信者のnoteと比較する表現(敵を作る)
- ❌ ライバル関係の人をブロックして引用(必ずバレる)
- ❌ 購入者の感想を無許可で転載

## 💡 投稿時間のベストプラクティス

30代後半〜40代 肌荒れ層に最も刺さる時間帯:

| 時間 | 特徴 | 向いている投稿 |
|---|---|---|
| 朝 7:00〜8:00 | 通勤・支度中にスマホを見る | 短い共感投稿・予告リマインド |
| 昼 12:00〜13:00 | 昼休みの情報収集 | スレッド・中身のチラ見せ |
| 夜 21:00〜23:00 | 最もエンゲージメントが高い | ローンチポスト・告知の本命 |

**ローンチポストは必ず 21:00〜22:00 に。** 朝や昼のローンチは伸びにくいです。

## 🎯 反応を最大化する小技

1. **スレッドは4本程度まで**(それ以上は読了率が落ちる)
2. **1本目は140字以内**(最もリポストされやすい長さ)
3. **引用リポストは"自分の気づき"を添える**(ただリポストするだけだと伸びない)
4. **コメントに2時間以内に返信**(アルゴリズムが"盛り上がっている投稿"と判定)
5. **販売期間中はリプ欄に"感想待ってます"を明示**(気軽に書いてもらえる)

## 🔁 購入者感想をもらいやすくする仕掛け

販売開始後、**noteの最後の文章**を以下のように締めると感想が集まります:

> このnoteが役に立ったら、感想を「#〇〇〇〇」でXに投稿してもらえると、すべて見に行きます。Before/Afterもぜひ。あなたの変化が、同じ悩みを持つ誰かの希望になります。

→ ハッシュタグでつぶやいてもらう → 引用リポストで次の購入者の後押しになる、というループ。

ハッシュタグ候補(覚えやすく短い):
- `#内側から肌改善`
- `#肌リセット30日`
- `#本当の原因`
