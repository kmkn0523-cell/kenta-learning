# @RoninWords — 完全投稿戦略ガイド

---

## アカウント概要

**コンセプト：** 日本の武士道・禅・格言を英語圏に届けるコンテンツブランド  
**ターゲット：** 欧米の25〜45歳男性（自己啓発・ミニマリズム・東洋哲学に関心あり）  
**差別化：** 翻訳ではなく「文化の文脈ごと届ける」深さ。カード画像の圧倒的な美しさ。

---

## プラットフォーム戦略（3軸運用）

| | X | Threads | TikTok / Reels / Shorts |
|---|---|---|---|
| **役割** | 文章でバズらせる（本拠地） | 画像で魅せる（Instagram連携） | 動画で広げる（新規流入） |
| **投稿頻度** | 2投稿/日（朝・夜） | 1投稿/日 | 3投稿/週 |
| **フォーマット** | 格言ツリー＋カード画像 | カード画像＋短文キャプション | 15〜30秒シネマティック動画 |
| **強み** | 拡散・リプライ文化 | 画像映え・Instagram流入 | アルゴリズム流入・バイラル |
| **文字数** | ツリー形式で実質無制限 | 500文字 | キャプションのみ |

---

## X 投稿フォーマット（詳細）

### 朝投稿 — 格言ツリー（8:00 AM EST）
```
Tweet 1: 漢字 + ローマ字（カード画像添付）
Tweet 2: 英訳（短く、詩的に）
Tweet 3: 文化的背景・ストーリー（3〜4文）
Tweet 4: 問いかけCTA + ハッシュタグ
↓ 投稿直後（手動）: 刺さる一言リプライ
```

### 夜投稿 — 一言ポスト（8:00 PM EST）
```
テキストのみ。シンプルに。
ハッシュタグは1〜2個まで。
問いかけより「断言」で終わる。
```

### ハッシュタグ戦略
- **メイン（毎回）：** `#JapaneseWisdom` `#Bushido`
- **ローテーション：** `#Samurai` `#Stoicism` `#Zen` `#Mindset` `#Japan`
- **格言固有：** `#Kintsugi` `#WabiSabi` `#Ikigai` など内容に合わせて

---

## Threads 投稿フォーマット（詳細）

### 基本テンプレート
```
[漢字] — [Romaji]

"[英訳]"

[1〜2文の解説。深く、短く。]

#JapaneseWisdom #Samurai #Bushido
```

### 画像仕様
- **サイズ：** 1080 × 1350px（縦型 4:5）※現在のカードは要最適化
- **枚数：** 1〜2枚（複数枚はカルーセル投稿でDay別格言まとめに使う）
- **フォント：** Yuji Syuku（漢字）／Cormorant Garamond（英語）

### Threads × Instagram 連携
- ThreadsはInstagramと同アカウントで連動
- Instagram Stories でカード画像を週1〜2回投稿
- Instagramプロフィールに「Daily Japanese Wisdom → @RoninWords」と記載

---

## TikTok / Reels / Shorts 戦略

### 投稿スケジュール
- **頻度：** 週3本（月・水・金 推奨）
- **尺：** 15〜30秒
- **構成：** 映像 + 日本語ナレーション + 英語字幕（台本30本済み）

### 素材調達（無料）
- Pexels / Unsplash → `samurai` `cherry blossom` `fog forest` `japan nature`
- 編集：CapCut（無料・スマホ可）
- BGM：Pixabay Music → `Koto` `Shakuhachi` で検索

### バイラル狙いのフック構成
```
0:00〜0:03 — 映像で即つかむ（桜散る・刀・霧の山）
0:04〜0:15 — 日本語ナレーション（静かで力強く）
0:15〜0:25 — 英語字幕で意味が届く
0:25〜0:30 — 黒画面に漢字フェードイン
```

---

## エンゲージメント戦略（毎日5分）

1. **朝投稿直後：** ツリーの最後にリプライを手動追加
2. **`#JapaneseWisdom` 検索：** 上位3〜5投稿にリプライ（「One thing I'd add...」形式）
3. **コメント返信：** 全件返す（フォロワー1万まで）
4. **引用RT活用：** 他アカウントの格言投稿を引用して独自視点を追加

## フォロワー成長ロードマップ

### Phase 1｜0 → 1,000（コンテンツ基盤）
- ひたすら投稿する。質より継続。
- Day 1〜30のX投稿スケジュールを完走する
- Threads・動画は余裕があれば

### Phase 2｜1,000 → 5,000（認知拡大）
- 動画投稿を週3本に増やす
- 「このアカウントをフォローすると毎日格言が届く」CTAを強化
- Day 100到達を大きく告知（節目コンテンツ）
- デジタル商品の最初の販売テスト（壁紙パックなど）

### Phase 3｜5,000 → 10,000（マネタイズ本格化）
- オーダーメイドDMサービス開始
- Patreon または Substack 開設
- Etsy でポスター・アートプリント販売開始
- 他の格言系・自己啓発系アカウントとコラボ

### Phase 4｜10,000+（ブランド確立）
- スポンサー投稿（日本関連ブランド）
- Shopify でグッズ販売（Printful連携）
- 電子書籍「100 Japanese Proverbs」出版（Amazon KDP）
- 企業向けカスタム格言サービス
