# @RoninWords — バズる動画台本の型

---

## なぜ今の台本はバズりにくいか

現在の台本の問題点：
- 映像の説明が細かすぎて「映像ありき」になっている
- ナレーションが「説明」になっている（感情を動かしていない）
- 最初の3秒でスクロールを止める仕掛けがない
- 視聴者が「自分ごと」として受け取れない

---

## バズる動画の構造（30秒版）

```
0:00〜0:03  【フック】  スクロールを止める
0:03〜0:10  【緊張】    「え、どういうこと？」と思わせる
0:10〜0:22  【核心】    格言の意味を「体験」させる
0:22〜0:27  【刺さり】  視聴者自身の話にする
0:27〜0:30  【余韻】    漢字フェードイン。何も言わない。
```

---

## フックの型（6種類）

台本ごとに以下のどれかを使う。

### 型A｜逆説フック
「常識と逆のことを言う」
```
【NAR】「日本では、負けることが勝つことだと教える。」
```
→ 「え？どういうこと？」で止まる

### 型B｜問いかけフック
「視聴者に直接問いを投げる」
```
【NAR】「あなたは今、本当に生きているか？」
```
→ 自分ごとになる。答えたくなる。

### 型C｜数字・事実フック
「具体的な数字や歴史的事実で始める」
```
【NAR】「700年前、侍はこう生きた。」
```
→ 信憑性と好奇心を同時に刺激する

### 型D｜沈黙フック
「最初の2秒、何も言わない」
```
【映像】霧の中の刀。完全な静寂。
【NAR】（2秒後）「…これが、侍の答えだ。」
```
→ 静寂が逆に止まらせる

### 型E｜共感フック
「視聴者が「自分だ」と思う状況から入る」
```
【NAR】「何度も立ち上がれない気がする夜がある。」
```
→ 感情的な共鳴から入る

### 型F｜衝撃映像フック
「映像だけで止まらせる」
```
【映像】スローモーションで桜が散る。音なし。
（ナレーションは3秒後から始まる）
```
→ 音声オフで見ている人も止まる

## ナレーションの書き方（3原則）

### 原則1：「説明」より「体験」
❌ 悪い例：「一期一会とは、茶道から来た言葉で、一生に一度の出会いを大切にするという意味です。」
✅ 良い例：「この会話は、二度と来ない。この食事も。この笑顔も。」

### 原則2：文を短く切る
❌ 悪い例：「侍は毎朝死について考えることで、今日という日を全力で生きていました。」
✅ 良い例：「侍は毎朝、死を考えた。　恐れるためではない。　今日を生きるために。」

### 原則3：最後は「問い」か「断言」で終わる
- 問い型：「あなたなら、どうする？」
- 断言型：「それが、侍の答えだ。」

---

## 英語字幕の書き方（3原則）

### 原則1：詩的に、短く
❌ `This is a Japanese proverb that means you should cherish every moment.`
✅ `This moment will never come again.`

### 原則2：改行で間を作る
```
Fall seven times.

Rise eight.

That's the whole philosophy.
```

### 原則3：最後の1行は「視聴者へのメッセージ」
```
You're not behind.

You're just between falls.
```

## 映像の選び方（シンプルな基準）

素材はPexelsで無料入手。以下のキーワードで検索：

| 格言のテーマ | Pexels検索ワード |
|------------|----------------|
| 立ち上がる・再起 | `person standing sunrise` `getting up slow motion` |
| 自然・無常 | `cherry blossom falling` `fog forest japan` |
| 静寂・集中 | `samurai meditation` `candle dark` |
| 水・柔軟性 | `water flowing rock` `rain slow motion` |
| 炎・鍛錬 | `fire forge` `blacksmith` |
| 山・旅 | `mountain path fog` `hiking alone` |

**映像選びの鉄則：**
- 最初のカットは「動き」があるもの（静止画は止まらない）
- スローモーション素材を優先する
- 人物の顔は映さない（誰でも投影できるように）

---

## BGMの選び方

| 雰囲気 | Pixabay検索ワード | 使うシーン |
|--------|-----------------|-----------|
| 静寂・余韻 | `koto ambient` | 物の哀れ・諸行無常 |
| 力強さ | `taiko drum` `japanese war` | 背水の陣・覚悟 |
| 希望・夜明け | `japanese sunrise` | 七転び八起き・継続 |
| 深み・禅 | `shakuhachi meditation` | 侘び寂び・間 |
| 緊張 | `japanese tension` | 虎穴・当たって砕けろ |

**ルール：**
- BGMは字幕が読める音量（声より小さく）
- 最後の3秒は音量を下げる（余韻を作る）

## 台本の書き換えテンプレート

以下の穴埋め形式で30本を書き換える。

```
【タイトル】DAY XX — [格言名]
【尺】約XX秒
【フック型】A〜Fのどれか

【映像】
0:00 - [フック映像。動きがあるもの]
0:03 - [緊張を高める映像]
0:10 - [核心を伝える映像]
0:22 - [静的。余韻を作る]
0:27 - 黒画面。漢字フェードイン。

【NAR】
（フック）「[型A〜Fのどれかを使った最初の1文]」
（核心）「[格言の意味を体験させる2〜3文。短く切る]」
（刺さり）「[視聴者への直接語りかけ]」

【SUB】
[英訳。詩的に。改行で間を作る]

[最後の1行：視聴者へのメッセージ]

【BGM】
[上の表から選ぶ]
```

## 改善例：DAY 03「七転び八起き」

### 改善前（現在）
```
【NAR】「倒れることは、恥ではない。立たないことが、恥だ。」
【SUB】Fall seven times. Rise eight.
【BGM】静かなピアノ。立ち上がる瞬間に弦楽器が加わる。
```

### 改善後（バズる型）
```
【フック型】型E（共感フック）

【映像】
0:00 - 泥の上に倒れた手のアップ。指が震えている。スロー。
0:04 - 手が、ゆっくりと地面を押す。
0:10 - 立ち上がる足元。夜明けの光。逆光のシルエット。
0:22 - シルエットが空を見上げる。風。静止。
0:27 - 黒画面。「七転び八起き」フェードイン。

【NAR】
「もう立てないと思った日が、あったはずだ。」

「侍は、転ばない訓練をしたのではない。
　転んだあと、誰より早く立つ訓練をした。」

「お前はまだ、立てる。」

【SUB】
They didn't train to never fall.

They trained to rise faster
than anyone expected.

You're not finished.
You're between falls.

【BGM】
最初：静寂のみ（環境音）
立ち上がる瞬間：低い弦楽器が1音だけ鳴る
終わり：消えるように
```
