# @RoninWords — Threads投稿テンプレート完全ガイド

---

## Threadsで「バズる」とはどういうことか

Threadsのアルゴリズムは以下の順でリーチを広げる：

| 指標 | リーチへの影響 | 狙い方 |
|------|------------|------|
| **コメント数** | 最強 | 「答えたくなる」投稿を作る |
| **シェア（引用・リポスト）** | 強い | 「保存して誰かに見せたい」内容にする |
| **いいね数** | 補助的 | 共感を呼ぶ締めの言葉 |
| **保存数** | じわじわ広がる | 「また読みたい」と思わせる深さ |

つまり **「コメントしたくなる投稿」** が最強。

---

## 投稿の基本構造（5パーツ）

```
[フック]          ← ここで止まるかどうか決まる（1〜2行）
                  ← 空行（必須）
[格言本体]        ← 漢字 / ローマ字 / 英訳
                  ← 空行
[体験させる解説]  ← 2〜3文。短く切る。説明しない。体験させる。
                  ← 空行
[刺さり]          ← 問いかけ or 断言（最後の1行が命）
                  ← 空行
[ハッシュタグ]    ← 3〜5個
```

---

## フックの型（6種類）

投稿ごとに以下のどれかを使う。

### 型A｜逆説フック
「常識と逆のことを言う」
```
The Japanese don't teach you to win.
They teach you to lose better.
```
→ 「え？どういうこと？」で止まる

### 型B｜問いかけフック
「視聴者に直接問いを投げる」
```
When did you last do something that truly scared you?
```
→ 自分ごとになる。コメントしたくなる。

### 型C｜数字・事実フック
「具体的な数字や歴史的事実で始める」
```
700 years ago, samurai had a word for this.
We still don't have a good English translation.
```
→ 信憑性と好奇心を同時に刺激する

### 型D｜沈黙フック
「何も説明せず、漢字だけで始める」
```
七転び八起き
```
→ 意味を知りたくて続きを読む

### 型E｜共感フック
「視聴者が「自分だ」と思う状況から入る」
```
You've been trying. It's still not working.
That feeling has a name in Japanese.
```
→ 感情的な共鳴から入る

### 型F｜衝撃事実フック
「知らなかった事実で始める」
```
Samurai were required to write poetry.
Not as a hobby. As survival.
```
→ 「え、そうなの？」で止まる

## 投稿フォーマット（4種類）

---

### フォーマット1｜画像型（毎日のメイン投稿）

書道カード画像 + テキスト。毎日の基本。

**テンプレート：**
```
[フック（型A〜Fから選ぶ）]

[漢字] — [Romaji]

"[英訳（詩的に、短く）]"

[体験させる解説 2〜3文。短く切る]

[問いかけ or 断言（最後の1行）]

#JapaneseWisdom #Bushido #[内容に合うタグ1個]
```

**実例（一期一会）：**
```
The Japanese have a word for this.
No direct English translation exists.

一期一会 — Ichi-go Ichi-e

"One time, one meeting."

Every conversation you've ever had
is gone forever.

This one too.

What would you say differently if you knew?

#JapaneseWisdom #Zen #Mindfulness
```

---

### フォーマット2｜テキストのみ型（夜投稿・深みを出す）

画像なし。文章だけで刺す。シンプルさが武器。

**テンプレート：**
```
[1行フック]

[格言の核心を3〜5行で表現。改行で間を作る]

[最後の断言 or 問いかけ]
```

**実例（武士道）：**
```
Samurai didn't meditate to feel calm.

They meditated to be
comfortable with discomfort.

There's a difference.
```

**実例（七転び八起き）：**
```
They didn't train to never fall.

They trained to rise faster
than anyone expected.

You're not finished.
You're between falls.
```

---

### フォーマット3｜問いかけ型（エンゲージメント狙い）

コメントを引き出すことが目的。週1〜2回使う。

**テンプレート：**
```
[問いかけフック（型B）]

[格言3つを並べる（選択肢を与える）]

[コメントを促すCTA（Drop ↓ など）]
```

**実例：**
```
Which one hits you harder right now?

一期一会 — "This moment will never come again."
七転び八起き — "Fall seven times. Rise eight."
侘び寂び — "Find beauty in imperfection."

Drop yours below. ↓
```

**別パターン（深掘り型）：**
```
One question. No wrong answer.

If a samurai lived your day today —
what would they change first?

Tell me. ↓
```

---

### フォーマット4｜ストーリー型（深みと権威を出す）

格言の背景・歴史・文化的文脈を語る。週1回。

**テンプレート：**
```
[衝撃事実フック（型F or 型C）]

[歴史的背景 or エピソード（3〜4文）]

[格言本体]

[現代への橋渡し（1〜2文）]

[締め（断言）]

#JapaneseWisdom #Samurai #[内容に合うタグ]
```

**実例（武士道）：**
```
A samurai general who never lost a battle
had this painted above his door.

Not a victory slogan.
Not a warrior's creed.

Just two characters.

武士道 — Bushido

"The way of the warrior is to die."

He didn't mean death.
He meant: commit fully, or don't start.

Most people start.
Few commit.

#Bushido #Samurai #JapaneseWisdom
```

## ハッシュタグ戦略

### 毎回使う（必須2個）
`#JapaneseWisdom` `#Bushido`

### ローテーション（1〜3個追加）

| タグ | 使うシーン |
|------|---------|
| `#Samurai` | 武士道・歴史系の格言 |
| `#Zen` | 禅・瞑想・内省系 |
| `#Stoicism` | 克己・鍛錬・覚悟系 |
| `#Mindset` | 自己啓発・成長系 |
| `#Ikigai` | 生きがい・目的系 |
| `#WabiSabi` | 不完全・美・受け入れ系 |
| `#Kintsugi` | 再起・傷・強さ系 |
| `#Mindfulness` | 今この瞬間・気づき系 |
| `#Japan` | 日本文化全般 |

**ルール：**
- 最大5個まで（多すぎるとスパム扱い）
- 格言の内容に合わないタグは使わない
- Threadsはハッシュタグの効果がXより弱いので、過信しない

## 投稿後アクション（毎日5分）

1. **投稿直後 → 5分以内：**
   `#JapaneseWisdom` で検索 → 上位3投稿にリプライ
   （「One thing I'd add to this...」形式が効く）

2. **コメントへの返信：**
   全件、24時間以内に返す（フォロワー1万まで）

3. **週1回（日曜夜）：**
   どのフォーマット・フックが一番伸びたか確認してメモ
