第1章 ― なぜスキンケアでは治らないのか

肌は「排出器官」だという視点

皮膚は、体の中で最大の臓器です。そして、体内の老廃物や炎症物質を排出する役割も担っています。

腸や肝臓の働きが落ちて、体内で処理しきれなかった毒素や炎症性物質は、汗や皮脂と一緒に皮膚から出ようとします。これが、内側の不調が"肌荒れ"という形で表面に現れるメカニズムです。

だから、いくら外から化粧水を塗っても、中で毒素が作られ続けている限り、皮膚はそれを排出しようとし続ける。これが「何をやっても治らない」の正体です。

肌荒れを起こす3大原因

体験と情報収集の結果、僕が行き着いた答えはこの3つです。

1. 皮脂分泌のコントロール失敗

糖質過多、ホルモンバランスの乱れ、睡眠不足、ストレスなどで、必要以上の皮脂が分泌される。皮脂自体は悪ではなく、"過剰に出る"状態が問題。過剰皮脂はアクネ菌のエサになり、毛穴の詰まり・炎症を引き起こします。

2. 腸内環境の悪化

小麦(グルテン)、食品添加物、精製糖、アルコールなどで腸粘膜にダメージが蓄積すると、リーキーガット(腸漏れ)と呼ばれる状態になります。これは、本来通さないはずの未消化物や毒素が腸壁を通過して血流に入ってしまう状態。体は全身で炎症反応を起こし、その一部が肌に出ます。

3. 慢性炎症(口腔・歯周含む)

見落としがちですが、口腔内の慢性炎症(歯周病・虫歯)は全身に微弱な炎症を広げます。歯周病菌が血流に乗って体内を循環することは、循環器領域の研究でも示されています。肌荒れが治らない人は、一度歯科でクリーニングと歯周検査を受けるだけで変化が出ることがあります(僕もそうでした)。

まず押さえるべき優先順位

この3つのうち、「皮脂」と「腸」を整えることが最優先です。口腔ケアは並行して、月1回の歯科クリーニングを習慣化すれば十分です。

第4章 ― 食事改革(デトックス編)

ここが本丸です。肌を変えるうえで、食事改革が最も効果が大きいと断言できます。

ステップ1:引き算(最初の2週間)

まず、以下の3つを可能な範囲で減らしてください。完全除去でなくてOKです。

① 小麦(グルテン)

パン、パスタ、うどん、ラーメン、餃子の皮、お好み焼き、天ぷらの衣、市販の洋菓子…。グルテンは腸粘膜を刺激し、リーキーガットの主因になります。

代替:米、米粉パン、十割蕎麦、グルテンフリー麺

② 精製糖

白砂糖、シロップ、菓子パン、清涼飲料水、ケーキ、アイス、チョコ菓子。血糖値スパイクを起こし、皮脂分泌を刺激します。

代替:はちみつ(少量)、果物(1日1個)、ダークチョコ(カカオ70%以上)

③ 食品添加物

ここはほとんどの人が知らない、スーパーで一番役に立つ知識です。

🔑 原材料表示の「／(スラッシュ)」を見てください

加工食品の裏面ラベルにある「原材料名」を見ると、途中に 「／」 という記号が入っています。

原材料名:小麦粉、砂糖、植物油脂、食塩 / 乳化剤、香料、着色料、保存料(ソルビン酸K)
                                    ↑
                                ここから後ろが全部「食品添加物」
  • 「／」より前 → 食材そのもの(小麦粉、砂糖、油、塩など)
  • 「／」より後ろ → 食品添加物(保存料、乳化剤、香料、着色料、人工甘味料など)

つまり、「／」より後ろが長い食品ほど、添加物まみれということ。腸内細菌を乱し、慢性炎症の原因になります。

🛒 スーパーで5秒で判別する方法
  1. 商品を裏返す
  2. 原材料名を見る
  3. 「／」が無い、または／以降が2〜3個まで → OK
  4. 「／」以降がズラッと長い → 棚に戻す

これだけで、肌に悪い加工食品の8割は避けられます。

⚠ 特に避けたい添加物
  • 人工甘味料:アセスルファムK、スクラロース、サッカリン(腸内細菌を乱す代表)
  • 保存料:ソルビン酸K、安息香酸Na
  • 着色料:赤色◯号、黄色◯号、カラメル色素
  • 増粘多糖類(具体名が書かれていない=複数混在のサイン)
💡 実例
食品 ／の前 ／の後
良い例:無添加豆腐 大豆、にがり (なし)
悪い例:カップ麺 小麦粉、植物油脂、醤油… 増粘剤、乳化剤、酸化防止剤、香料、着色料、酸味料、調味料(アミノ酸等)…

「／の前だけで成り立っている食品」を選ぶ――これがスーパーで迷わない最強のルールです。コンビニよりスーパーの惣菜コーナー、加工品より素材そのもの。

健康診断で「貧血なし」と言われても安心できない

でもこれは鉄不足の最終段階でしか引っかからない指標です。

体内の鉄は次の順番で減ります：

  1. 貯蔵鉄（フェリチン）が減る ← ここが減っていても健診では分からない
  2. 貯蔵鉄が枯渇する
  3. ようやくヘモグロビンが減る ← 健診で「貧血」と診断

つまり、ヘモグロビン正常でもフェリチンが枯渇している人は山ほどいる。これが「潜在性鉄欠乏（隠れ貧血）」です。

第7章 ― 睡眠とホルモンの整え方

肌を作っているのは、寝ている時間です。ここを疎かにすると、他をどれだけ頑張っても成果が出ません。

ゴールデンタイムの真実

「22時〜2時が肌のゴールデンタイム」は、実際には "就寝してから最初の90〜180分" が正解です。この時間帯に成長ホルモン分泌のピークが来ます。

つまり、夜11時に寝ても、深夜2時に寝ても、「寝始めの深い睡眠をしっかり取る」ことが鍵。ただし、体内時計の観点から 0時を大きく超えるほど皮膚の修復効率は落ちます。できれば23時台、遅くとも0時には布団に入りたい。

睡眠の質を上げる7つの習慣

  1. 朝日を3〜5分浴びる(起床後30分以内) → メラトニンが夜に出るよう、体内時計をリセット

  2. カフェインは14時まで → コーヒーの半減期は約5時間。15時のコーヒーは夜まで残る

  3. 夕食は就寝3時間前までに → 消化中は深い睡眠に入れない

  4. 就寝1時間前はスマホ・PCを閉じる → ブルーライトがメラトニン分泌を抑制

  5. 寝室の温度は18〜20度、湿度50〜60% → 深部体温が下がると眠くなる仕組み

  6. 寝る前のアルコールを避ける → 寝付きは良くなるが睡眠の質は大幅に低下

  7. ベッドは"寝る専用"にする → ベッドでスマホ・仕事をしない。脳が「ベッド=寝る場所」と認識する

💬 僕が変化を感じた順番

  • 朝日を浴び始めた → 3週間後、寝付きが良くなった
  • 寝る前のスマホをやめた → 翌朝のテカりが減った
  • カフェインを14時までに制限 → 深い睡眠の時間が伸びた

小さな変化の積み重ねです。いきなり全部やろうとせず、1つずつ足していくのがコツ。

第8章 ― ストレスと肌の関係【新規】

ストレスが肌を荒らすメカニズム

ストレスを感じると、副腎からコルチゾールというホルモンが分泌されます。これが慢性的に出続けると:

  • 皮脂分泌が増える(テカり・ニキビ悪化)
  • 肌のバリア機能が低下(敏感肌化)
  • 睡眠の質が落ちる(成長ホルモン低下)
  • 腸の蠕動運動が乱れる(便秘・下痢)

つまり、ストレスは肌荒れのあらゆる原因を同時にONにするスイッチのようなものです。

第9章 ― 口腔ケアと肌【新規】

第1章で「肌荒れの3大原因」の1つに慢性炎症（口腔・歯周含む）を挙げました。これは多くの人が見落としているけれど、慢性肌荒れの治らない原因のトップ3に入る盲点です。

「歯磨きと肌、本当に関係あるの？」と思った人ほど、この章は読む価値があります。

口の中の細菌は、血流に乗って全身を回る

歯周病で歯茎が炎症している状態では、歯周病菌が血管から血流に侵入します。これを医学用語で「菌血症（きんけつしょう）」と呼びます。

毎日の歯磨き・食事のたび、歯周病がある人の血液には細菌が流れ込み、全身を巡る。免疫がこれを攻撃→全身慢性炎症→肌に出る。

研究結果を並べると、歯周病患者は：

  • 心筋梗塞リスク 1.4倍
  • 糖尿病リスク 2倍
  • 認知症リスク 1.7倍
  • 早産・低体重児リスク 3倍
  • 動脈硬化進行 1.5倍

これらは循環器・内分泌・脳神経・産科などのさまざまな分野で報告されています。肌荒れがリストに加わっても不思議じゃありません。

「スキンケアより歯科クリーニングが効いた」は嘘じゃない

僕の体験談：

慢性的に頬とフェイスラインの炎症が引かない時期がありました。食事も整え、サプリも飲み、睡眠も整えた。それでも完全に治らない。

ふと思い立って3年ぶりに歯科検診を受けたら、軽度の歯周病と判明。

スケーリング（歯石除去）と歯周ポケット清掃を3回受けた1ヶ月後、長年悩んでいたフェイスラインの赤みが明らかに引きました。

「スキンケアより歯磨きが効いた」――今ではこれを冗談ではなく真顔で言います。

歯科受診の最適頻度

状態 受診頻度
健康な歯茎 半年に1回
歯周病既往あり 3〜4ヶ月に1回
歯肉炎・出血あり すぐ受診
長年検診してない まず1回行く

定期検診は保険適用で3,000〜5,000円。スケーリング込みで1回1時間程度。

「美容皮膚科に1回3万円払う前に、歯科クリーニング1万円分やれ」と心から言いたい。

自宅でできる口腔ケア5つ

1. 歯磨きは「磨き残し」をなくす技術

時間より方法。1日2回、2分以上ではなく全部位を磨くこと。

  • 歯と歯茎の境目に45度で当てる
  • 1本ずつ細かく動かす（5〜10mmずつ）
  • 力は弱めに（毛先が広がらない強さ）

電動歯ブラシ（フィリップス Sonicare、ブラウン Oral-B）があれば技術が補正されます。1万円台で買えて、肌への投資としては圧倒的に安い。

2. デンタルフロス（毎日必須）

歯磨きだけでは歯間の汚れの30〜40%しか取れません。残り60〜70%はフロスで取る。

選び方： - ワックスタイプ：滑りやすく初心者向け - ノンワックス：汚れを絡め取る上級者向け - Y字フロス：奥歯にも届きやすい、最初はこれ

夜寝る前に1回。1日たった2〜3分で口腔細菌を激減できる。

3. 舌ブラシで舌苔を除去

朝起きたとき、舌が白い人は要注意。舌苔（ぜったい）は細菌の温床で、口臭・全身炎症の原因。

  • 専用舌ブラシ（ピジョン・SHIKIEN W-1等、300〜800円）
  • 朝1回、舌の奥から手前へ3〜5回だけ
  • 強くこすらない（味蕾を傷つける）
  • 歯ブラシで代用するのはNG（毛が硬すぎる）

4. マウスウォッシュは「ノンアルコール」を選ぶ

アルコール入りは口腔内を乾燥させ、逆に細菌が増えやすくなる。

おすすめ製品（ノンアルコール）： - コンクールF（薬用、歯科推奨多数） - ガム・デンタルリンス（敏感肌タイプ） - リステリン トータルケアゼロ（ノンアルコール版）

夜の歯磨き＆フロスの後、20mlを30秒間ブクブクして吐き出す。

5. 口呼吸をやめる

口呼吸は口腔内の乾燥→唾液による細菌除去ができない→細菌増殖。寝ている間ずっと口呼吸の人は、リカバリーが効きません。

対策： - 意識的に鼻呼吸を訓練（日中、口を閉じる） - 鼻づまりは耳鼻科で治療（アレルギー性鼻炎・副鼻腔炎） - 寝るときは「マウステープ」（ネルネル等、薬局・Amazonで500円〜） - 加湿器で寝室の湿度50〜60%（鼻腔を乾燥させない）

「口→腸→肌」の繋がり

口腔内の悪玉菌は飲み込まれて腸に到達します。歯周病菌の代表「ジンジバリス菌」は、

  • 腸内環境を悪化させる
  • 腸のバリア機能を弱める（リーキーガットを悪化）
  • 大腸がんリスクを高める

…ことが研究で確認されています。

逆ルートも成立：腸内環境が悪い人は唾液中の免疫力（IgA）も落ちて、口腔内も荒れやすい。

口と腸はワンセットで考えるのが正解。第5章のリーキーガット対策と、この章の口腔ケアを並行することで効果が倍増します。

  • 歯科クリーニング＋自宅口腔ケア1ヶ月：残り3割が消えた

最後のピースを埋めたのが口腔ケアでした。やる順番は最後でもいいけれど、やらないと完成しない。

この章のまとめ

  • まず歯科で検診を受ける（3,000〜5,000円、半日）

第10章 ― 外側ケアの引き算

ここまで内側の話ばかりしてきました。外側ケアをまったくするな、という話ではありません。"最小で最大の効果"を出すが結論です。

僕が行き着いた外側ケアのミニマリズム

朝

  • ぬるま湯洗顔のみ(洗顔料は使わない)
  • 化粧水1本

夜

  • 低刺激の洗顔料
  • 化粧水1本

以上です。

引き算の理由

過度な保湿・高機能美容液を塗りすぎると、皮膚は「十分に保湿されている」と勘違いして、自分で皮脂を出す機能を弱めます。すると、スキンケアをやめた瞬間に乾燥が悪化する"スキンケア依存肌"になります。

内側を整えながら、皮膚本来の保湿力を取り戻すのが正解。

洗顔料の選び方

  • 成分表示がシンプル
  • アルコール(エタノール)不使用
  • 合成界面活性剤が強すぎない(ラウリル硫酸Naは避ける)
  • 弱酸性〜中性

候補としては、キュレル、ミノン、イハダ、NOVなどの敏感肌向けライン。僕はキュレルを数年使っています。

化粧水の選び方

  • 無香料・無着色
  • アルコールフリー
  • 余計な"美容成分"が山盛りでない

「これ1本で美白・シワ・たるみ・保湿…」みたいな多機能品は、刺激成分も多いことが多いです。

肌断食を試す価値

慢性的な肌荒れが続いている人は、週1日だけ"何も塗らない日"を作ってみてください。夜の洗顔後、化粧水も塗らずに寝る。

最初は乾燥して気持ち悪いですが、皮膚の自立機能を取り戻すのに効きます。いきなり365日やる必要はなく、週1からでOK。

Q1. 効果が出るまでどれくらいかかりますか?

A. 1日20分の散歩で十分。激しい筋トレは活性酸素を増やして逆効果になることも。

A. 週1〜2回なら有益。汗による排出、自律神経調整に効く。ただし、肌が炎症期のときは避ける。

A. 低刺激のミネラルファンデ推奨。帰宅後はすぐ落とす。1日中厚塗りは肌にとって負荷。
