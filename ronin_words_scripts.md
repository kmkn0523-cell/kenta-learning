# @RoninWords — ショート動画台本 30本
# 尺：15〜30秒 / スタイル：シネマティック
# 構成：映像 + 日本語ナレーション + 英語字幕
# プラットフォーム：TikTok / Instagram Reels / YouTube Shorts
# ────────────────────────────────────────────────────────

## 台本の読み方
- 【映像】 → カメラに映すもの・動き
- 【NAR】  → 日本語ナレーション（読み上げテキスト）
- 【SUB】  → 英語字幕（画面に表示）
- 【BGM】  → 音楽の方向性
- ※ 尺はナレーションのペースで調整

## 共通制作メモ

### 映像素材の方向性
- **メイン素材：** Pexels / Unsplash（無料）で「samurai」「japan nature」「fog forest」「cherry blossom slow motion」で検索
- **補完：** CapCut / DaVinci Resolveでスローモーション加工
- **テキストアニメ：** CapCut のテキストエフェクト「フェードイン」「縦書き」

### 字幕スタイル
- フォント：セリフ系（Cormorant Garamond 推奨）
- 色：白、または薄いクリーム色
- 位置：画面下20%
