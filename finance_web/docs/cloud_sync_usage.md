# クラウド同期 使い方ガイド（BYB）

スマホとPCなど複数の端末で家計データを同期する機能の使い方と仕組みです。
設計の詳細は `finance_web_sync_design_v2.md` を参照してください。

## いちばん大事なこと（先に読む）

- **復元キー（RK）** … データを取り戻すための長い文字列。**最初の1回だけ表示され、二度と表示されません**。パスワード管理アプリやメモに必ず保管してください。
- **端末パスワード** … その端末で同期を再開するための合言葉（4文字以上）。端末ごとに自由に決められます。
- **復元キーと、それを使った端末のパスワードを両方失うとデータは復元できません。** サーバ管理者でも中身を読めない（後述）ため、誰も復旧できません。

## 1台目で同期を始める

1. 画面下の `⚙️ 設定` を開く
2. 「☁️ クラウド同期」の **「同期を始める（新しい復元キーを作る）」** を押す
3. 表示された **復元キーを保管**する（コピーボタンあり）
4. 「復元キーを安全な場所に保管しました」にチェック
5. **この端末用のパスワード**（4文字以上）を入力
6. 「この端末で同期を開始」を押す → 状態が「同期中（待機）」になれば成功

## 2台目（別の端末）で参加する

1. 2台目で同じアプリを開き、`⚙️ 設定` → 「☁️ クラウド同期」
2. **「復元キーで別端末から復元」** を押す
3. 1台目で保管した **復元キー** を入力
4. **この端末用のパスワード**（2台目用。1台目と同じでなくてよい）を入力
5. 「復元する」を押す → 1台目のデータが取り込まれます

## アプリを開き直したとき（再開）

- アプリを閉じて開き直すと、同期は **「ロック中」** になります
- `⚙️ 設定` で **その端末のパスワードを入力 → 「同期を再開する」** で再開します
  （復元キーの再入力は不要。パスワードだけでOK）

## 設定が食い違ったとき（競合の解決）

予算・カテゴリ設定・貯金目標を2台で別々に変えると「競合」になります。

- 設定画面に **⚠️ どちらを残すか選ぶ画面** が出ます
- 各項目で「この端末を残す」か「別端末を残す」を選べばOK
- 取引・収入などの **一覧データは競合しません**（両方の変更が自動でマージされます）

## 仕組み（安心のために）

- データは **端末側で暗号化してから** サーバ（Vercel KV / Upstash）に送ります。
  サーバは中身を一切読めません（**E2E暗号化**）。
- 復元キーから「アカウントID」と「暗号鍵」を計算で導出するため、
  同じ復元キーを使えばどの端末でも同じデータを復号できます。
- 約30秒ごとに自動で同期し、データを変えると5秒ほどで送信されます。

## うまくいかないとき

- **「パスワードが違います」** … 端末パスワードの入力ミス。正しいパスワードを入れ直す
- **状態が「エラー」** … 通信不良の可能性。時間をおいて「今すぐ同期」を押す
- **短時間に何度も操作してエラー** … 連続アクセスの保護が働くことがあります。少し待って再試行
- **復元キーを無くした** … 残念ながら復旧できません。今動いている端末のデータをバックアップ（設定のエクスポート）してから、新しい復元キーで作り直してください
