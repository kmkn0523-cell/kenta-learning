### 1. なぜ自作したか（300字）

#### 既存アプリへの不満
- マネーフォワード/Zaim: 銀行連携が魅力的だが、データを企業に渡すのが嫌
- 無料版は広告ベタベタ＋機能制限
- 月数百円のサブスクは積もると年5,000円超

#### 自作のメリット
- データは自分の端末内で完結（プライバシー100%）
- 機能は自分で必要なものだけ実装できる
- 無料で家族にも配れる
- 学習にもなる

#### 機能（v1.1.0時点）

### 3. 技術スタック（300字）

- フロント: React 18 + TypeScript + Vite
- グラフ: Recharts
- 暗号化: WebCrypto API (AES-GCM + PBKDF2)
- ホスティング: Vercel（無料枠）
- データ保存: ブラウザ localStorage（暗号化済JSON）

> 「サーバなし」の設計が **データプライバシー** と **無料運営** を両立させる肝。

### 4. 苦労した点（300字）

- 暗号化キーの管理: パスワードからAES鍵を派生させて、メモリにだけ保持
- 月次データの集計: 月跨ぎ・年跨ぎの計算で苦戦
- バンドルサイズ: 初期595KBあったので React.lazy で **52KB に削減**
- TypeScript型定義: 永続化フックの型が複雑になりがち

### 5. マネタイズ計画（400字）

#### Phase 1 (現在)
- 完全無料公開
- 投資・ローン関連アフィリエイト（A8.net 4件契約済）
- 目標: ユーザー数50人

#### Phase 2 (2026/08〜)

#### Phase 3 (2026/12〜)
- **家計診断PDF出力** や **AI家計アドバイス** を有料化
- 目標: 課金100人 → 月¥50,000

### 6. PWAインストール方法（200字）

#### iPhone
1. Safariで https://financeweb-nine.vercel.app/ を開く
2. 共有ボタン □↑ → 「ホーム画面に追加」

#### Android
1. Chromeで同じURLを開く
2. メニュー → 「アプリをインストール」

ホーム画面にアイコンが追加され、ブラウザ起動不要でアプリのように使えます。
