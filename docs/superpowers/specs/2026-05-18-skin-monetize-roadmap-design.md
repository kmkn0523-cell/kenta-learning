## 1. 前提と目標

### 確定済みの前提
- **目標**: 2026/11/17時点で月3〜5万円の売上を作る
- **主軸**: Threads（既に1日4回自動投稿稼働中）
- **副軸**: Instagram（@skin_reset_jp、Batch 1 リリース済・自動継続）
- **マネタイズ対象記事**: 既存 note 原稿7本（skincare / sleep / food / food_premium / chokatsu / fasting / men_premium）
- **リソース投下**: 本業並み（平日2時間以上・週末3時間以上）。他プロジェクト（@RoninWords・finance_web・Python学習）は一部減速可

### 採用アプローチ
**A. 王道型「土台 → 集客 → 販売」直列**
- プロフィール再設計（M1）→ 毎日リプ＋A/B検証（M2）→ 先行 note リリース（M3）→ 通常価格＋X追加（M4）→ 上乗せ枠導入（M5）→ 定常化（M6）

### 不採用
- B. Threads+X 2正面早期マネタイズ（X追加は M4 に後置）
- C. データドリブン徹底（観測待ちが半年と相性悪い）

- M1-M2 は売上ゼロ前提。土台ができるまで売上は数えない

## 4. KPIと検証ループ

### リード指標（先行・日次〜週次で動く）
- 1日の能動アクション数（リプ・コメント返し）
- 1日の手動投稿数（自動投稿はカウント外）
- 投稿あたり views / likes / replies（`skin_analytics.json` で自動収集）
- プロフィールクリック数（Instagram、要 `instagram_manage_insights` 権限）

### ラグ指標（遅行・月次でしか動かない）
- 月末フォロワー数
- 月の合計 engagement_score（`skin_analyzer.py` で算出）
- 月の売上（note・アフィリ・メンバーシップ合算）
- A/B Batch の勝率（`skin_ab_report.py`）

### レビューサイクル

| 頻度 | やること | 所要時間 |
|---|---|---|
| 毎日（夜） | リプ件数・投稿数を `skin_daily_log.md` にメモするだけ。数字は見ない | 3分 |
| 毎週月曜朝 | 先週の views 平均・engagement_score 推移を確認、伸びた投稿を記録、今週試す型を1つ決める | 15分 |
| 毎月末 | フォロワー増・売上をフェーズ目標と照合。未達なら次月の打ち手を決める | 30〜45分 |

### 既存の測定基盤
- `skin_data_collector.py`（Threads API データ取得・GitHub Actions稼働中）
- `skin_analyzer.py`（型別ランキング算出）
- `skin_optimizer.py`（最適化インデックス生成）
- `skin_ab_report.py`（A/B 勝者判定）
- 追加で M2 に `skin_monthly_report.py`（月次サマリー）を新規作成

## 6. 撤退ライン・軌道修正の判断基準

### 3階層の判断ルール
- L1: 軽い未達（目標80%以上達成） → そのまま継続
- L2: 重い未達（50〜80%） → 戦略の一部を変更
- L3: 完全未達（50%未満） → 主軸そのものを見直す

### 月次チェックポイント

| チェック日 | フォロワー目標 | 売上目標 | L2 ライン | L3 ライン |
|---|---|---|---|---|
| 2026/06/17（M1末）| 30〜80 | 0円 | 20人未満 → プロフィール再診断、リプ件数2倍化 | 10人未満 → アカウント設計から疑う・別ペルソナ検討 |
| 2026/07/17（M2末）| 100〜200 | 0〜2,000円 | 60人未満 → 投稿型を3つに絞り、伸びた型を連投 | 30人未満 → ペルソナ・ジャンル再定義 |
| 2026/08/17（M3末）| 220〜380 | 1〜2万円 | note 2部未満 → 価格1,480円に下げる or LP差し替え | note 0部 ＋ フォロワー150未満 → ジャンル撤退検討 |
| 2026/09/17（M4末）| 370〜580 | 1.5〜2.5万円 | X views平均10未満 → X停止しThreads回帰 | 売上1万円割れ＋伸び鈍化 → 第2note前倒し or アフィリのみへ縮退 |
| 2026/10/17（M5末）| 520〜700 | 2〜3.5万円 | 上乗せ枠が動かない → メンバーシップ⇄相談で形態変更 | 売上2万円未満 → メンバーシップ廃止し、note 新刊リリース切替 |
| 2026/11/17（M6末・最終判定）| 700〜1,000 | 3〜5万円 | 売上2.5万円台 → 来期継続、軽微なテコ入れ | 売上1万円未満＋フォロワー400人未満 → 半年成果ゼロ判定、最終分岐へ |

### M3末 L3：「ジャンル撤退」検討の中身
- 既存資産は「肌荒れ」固有だが、腸活・睡眠・食事は隣接ジャンルとして流用可能
- 例: `@skin_reset_jp` → `@gut_reset_jp`（腸活軸）に再定位、肌荒れは派生テーマ扱い
- M3末で実行する場合、M4 はリブート月。M6 の目標は1〜2ヶ月後ろ倒し（半年→7〜8ヶ月）

### M6末 L3：最終分岐（半年で売上1万円未満）
3つから1つを必ず選ぶ。「もう1ヶ月だけ」は禁止。
1. 継続＋目標下方修正: 月1〜2万円目標で1年プランに切替
2. アカウント休眠＋資産売却: 自動投稿だけ残し、note は500円でロングテール
3. 撤退＋ナレッジ転用: 知見を `finance_web` or `@RoninWords` のマネタイズに転用

### 撤退判断の防御線
- 数字で決める（気合いではなく指標で判断）
- 「来月から本気出す」は自動的に L3 扱い
- 撤退は失敗ではない。未練で1年溶かす方が損失
