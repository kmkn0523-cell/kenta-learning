```markdown
# 肌荒れが治らない人の多くに、亜鉛不足があった。

「ビタミンCもコラーゲンも試したのに、なんか変わらない」

そういう人に一番多く欠けているのが、亜鉛です。

肌の材料として最も重要なミネラルのひとつなのに、現代の食事では慢性的に不足しやすい。ファストフード中心・白米中心の食生活だと、ほぼ確実に足りていません。

---

## 亜鉛が肌に必要な理由

亜鉛は「皮膚の構造タンパク質（コラーゲン）を作る」「皮脂の分泌量を調整する」「炎症を抑える」という3つの働きを持っています。

## 何を食べれば補えるか

亜鉛が豊富なのは牡蠣、牛赤身肉、豚レバー、大豆製品（納豆・豆腐）、カシューナッツなど。

サプリで補う場合は「亜鉛単体」か「亜鉛＋銅」のもの（過剰摂取で銅が不足するため）。1日15〜30mgが目安です。

外から塗るものを変えても治らない場合、材料不足が根本原因であることがほとんどです。

化粧水・美容液はあくまで「保湿の補助」。肌を作るのは体の内側からの栄養です。亜鉛不足を放置したまま外側ケアだけ続けても、バケツに穴が空いたまま水を注ぎ続けるようなものです。

---
```

```markdown
# ストレスで肌が荒れるのは、気のせいじゃなかった。

コルチゾールは血液を通じて全身に作用します。外から塗るものでは干渉できません。

「ストレスを感じたらいつもより丁寧にスキンケアする」という発想は、原因に対してアプローチできていない。内側のホルモンバランスを整えることが先決です。

このnoteではストレスが肌に与える影響と、コルチゾールを下げる生活習慣・サプリの具体的な使い方を詳しく解説しています。
```

```markdown
# 「甘いもの食べると肌が荒れる」は本当だった。そのメカニズム。

「チョコを食べると翌日ニキビが出る」という経験、持っている人は多いと思います。

これは迷信でも気のせいでもありません。砂糖と肌荒れには、血糖値を介した明確なメカニズムがあります。

---

## 血糖値スパイクが肌を壊す

砂糖や精製された炭水化物を食べると、血糖値が急上昇します（血糖値スパイク）。これを下げるためにインスリンが大量に分泌される。

このインスリンが皮脂腺を刺激して皮脂分泌を増やし、肌の角質ターンオーバーを乱します。さらに「AGEs（終末糖化産物）」という物質が体内で作られ、コラーゲンを硬化・分解させる。毛穴が詰まりやすくなり、肌のハリも失われていきます。

## 「何を」じゃなく「どう食べるか」が重要

砂糖を完全にやめる必要はありません。重要なのは血糖値の上昇スピードをゆるやかにすること。

対策は「食物繊維を先に食べる（ベジタブルファースト）」「白米・パンより玄米・全粒粉を選ぶ」「甘いものは食後に食べる（空腹時に単体で食べない）」の3つ。これだけで血糖値の急上昇を大幅に抑えられます。

## 人工甘味料も要注意

砂糖の代わりに使われる人工甘味料（アスパルテーム・スクラロースなど）は、腸内細菌に悪影響を与えるデータがあります。腸が乱れると肌に出る。砂糖より安心とは言い切れません。

---

このnoteでは血糖値コントロールのための具体的な食事の順番・選び方・置き換えリストを詳しく解説しています。
```

```markdown
# 肌荒れの人の多くが、水をほとんど飲んでいなかった。

「化粧水で保湿してるのに肌が乾燥する」

これ、外から補うだけでは追いつかない状況になっているサインです。体の内側、つまり飲む水が足りていない可能性があります。

---

## 肌の水分は「飲む」ことでしか補えない

肌の最も深い層（真皮層）にある水分は、外から塗るものでは届きません。化粧水が届くのはあくまで表皮の浅い部分。真皮のヒアルロン酸に水を届けるには、血流を通じて内側から補給するしかありません。

水分が不足すると、肌のターンオーバーが乱れ、古い角質が剥がれにくくなり、毛穴が詰まりやすくなる。保湿クリームを塗っても治らない乾燥ニキビの多くが、水分不足起因です。

## 1日に必要な水の量

体重×30mlが目安。体重60kgなら1日1,800ml。これは食事に含まれる水分も含むので、飲料としては1,200〜1,500mlが現実的な目標です。

コーヒー・アルコールには利尿作用があるため、飲んだ分だけ追加で水を飲む必要があります。「コーヒー1杯飲んだら水をコップ1杯追加」が覚えやすいルールです。

## 飲み方も重要

一気飲みは体が吸収しきれず、尿として出てしまいます。15〜30分おきにコップ半分ずつ飲むのが理想。起床直後と入浴前後は特に補給が重要なタイミングです。

---

このnoteでは水分補給のルール・タイミングと、飲み物の選び方（水vs白湯vsルイボスティーなど）を含む内側ケアの全体像を解説しています。
```

```markdown
紫外線（特にUVA）は肌の真皮層まで届き、コラーゲンを分解する酵素を活性化させます。さらに活性酸素を発生させ、細胞を酸化ダメージにさらす。これが慢性的な炎症の原因になります。
```

```markdown
## オメガ6過多・オメガ3不足の現代食

人間の体には「炎症を起こす物質（オメガ6由来）」と「炎症を抑える物質（オメガ3由来）」のバランスが必要です。

現代の食事はサラダ油・コーン油・大豆油を大量に使っており、オメガ6が圧倒的に多い。反対にオメガ3（青魚・亜麻仁油など）は少ない。この比率が崩れると、体が慢性的な炎症状態になります。
```

```python
# release_note.py
# note_queueフォルダから次の記事を取り出して note_ready.md に書き出すスクリプト
# GitHub Actions が毎週月曜 9:00 JST に自動実行する

import os        # ファイルの存在確認に使う道具
import json      # JSONファイルを読み書きする道具
import sys       # エラー時に終了コードを返す道具
from datetime import datetime  # 今日の日付を取得する道具

# ファイルのパス設定（このスクリプトがある場所を基準にする）
BASE_DIR      = os.path.dirname(__file__)                          # skinフォルダ
QUEUE_DIR     = os.path.join(BASE_DIR, "note_queue")              # 記事ストックフォルダ
PROGRESS_FILE = os.path.join(BASE_DIR, "note_queue_progress.json") # 進捗記録
READY_FILE    = os.path.join(BASE_DIR, "note_ready.md")            # 今週の記事


def load_progress():
    """進捗ファイルを読み込む（なければ0から開始）"""
    if not os.path.exists(PROGRESS_FILE):
        return {"next_index": 0}
    try:
        with open(PROGRESS_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    except (json.JSONDecodeError, FileNotFoundError):
        return {"next_index": 0}


def save_progress(next_index):
    """次に出す記事番号をファイルに保存する"""
    with open(PROGRESS_FILE, "w", encoding="utf-8") as f:
        json.dump({"next_index": next_index}, f, ensure_ascii=False, indent=2)


def get_article_files():
    """note_queueフォルダにある記事ファイルを番号順で取得する"""
    if not os.path.exists(QUEUE_DIR):
        return []
    files = sorted([
        f for f in os.listdir(QUEUE_DIR)
        if f.startswith("article_") and f.endswith(".md")
    ])
    return files


def main():
    today = datetime.now().strftime("%Y/%m/%d")
    print(f"=== note週次リリース開始 ({today}) ===")

    # 記事ファイル一覧を取得する
    article_files = get_article_files()

    if not article_files:
        print("⚠️  note_queueフォルダに記事がありません。スキップします。")
        return  # エラーにはしない（GitHub Actionsを失敗させない）

    # 進捗を読み込んで次の記事を決める
    progress = load_progress()
    next_index = progress["next_index"] % len(article_files)  # 30本使い切ったら最初に戻る
    article_filename = article_files[next_index]
    article_path = os.path.join(QUEUE_DIR, article_filename)

    print(f"📄 今週の記事: {article_filename} ({next_index + 1}/{len(article_files)})")

    # 記事ファイルを読み込む
    try:
        with open(article_path, "r", encoding="utf-8") as f:
            article_content = f.read()
    except FileNotFoundError:
        print(f"❌ 記事ファイルが見つかりません: {article_path}")
        sys.exit(1)

    # note_ready.md に書き出す（ヘッダーを付加する）
    header = f"<!-- 自動生成: {today} | {article_filename} -->\n\n"
    output_content = header + article_content

    with open(READY_FILE, "w", encoding="utf-8") as f:
        f.write(output_content)

    print(f"✅ note_ready.md を更新しました")
    print(f"   → skin/note_ready.md を開いてnoteにコピー&ペーストしてください")

    # 進捗を更新する（次回は次の記事）
    save_progress(next_index + 1)

    # 次回の記事を予告する
    next_next_index = (next_index + 1) % len(article_files)
    next_article = article_files[next_next_index]
    print(f"📅 来週の記事: {next_article}")
    print("=== 完了 ===")


if __name__ == "__main__":
    main()
```
